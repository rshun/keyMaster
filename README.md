# myKey

## 一个自用的密码管理器  

- 只要记住主密码，所有的密码都可以自动生成  

- 配置文件内容使用AES256方式加密保存，该密码建议和主密码不同。  

- 密码长度类型均自定义  

- 支持多用户  

- 无需联网  

### 第一次使用的时候  
执行以下命令导出模板配置文件
```
keylist -g demo
```
填写内容后执行以下命令
```
keylist -r 用户名 -a demo
```
输入配置文件密码即可。

### 正常使用
```
myKey 用户名 网站关键字
```
输入模板密码和主密码即可生成该网站的密码
